# Shelflife Inventory Management System

## 📌 Overview
Shelflife is a software system designed to track product expiration dates and manage inventory efficiently. The goal is to help reduce waste, improve organization, and provide users with clear visibility of product lifecycles.

This project demonstrates practical software development skills by solving a real-world problem through structured data handling and system design.

---

##  Features
-  Product inventory tracking  
-  Expiry date monitoring  
-  Alerts for expiring products (if implemented / planned)  
-  Organized data management  
-  Simple and user-friendly system logic  

---

##  Technologies Used 
- Kotlin Coding 
- Android Studio

---

##  Purpose of the Project
The purpose of Shelflife is to demonstrate:
- Problem-solving using software development  
- The ability to design and structure a functional system  
- Understanding of real-world applications (inventory management and waste reduction)  

This project specifically addresses food wastage in student residences. Many students struggle to keep track of expiration dates, which often leads to food being thrown away unnecessarily. As a result, students end up spending more of their allowance replacing items that could have been used before expiring.

Shelflife was developed to help students monitor their food items more effectively, reduce waste, and make smarter use of their limited budgets.

---

##  How It Works
1. User Signs up or Logs in
2. Users scans product barcode or manually enters it to add products into the system  
3. Each product includes an expiration date  
4. The system tracks and monitors these dates  
5. Alerts or indicators show when products are close to expiring
